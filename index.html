⚙️ Konfigurasi Awal Diperlukan

Aplikasi belum terhubung ke server penyimpanan. Buka file index.html, lalu isi bagian APP_CONFIG di awal kode dengan kredensial proyekmu. Ikuti panduan lengkap pada file SETUP.md.

1
Baca file SETUP.mdIkuti tutorial langkah demi langkah untuk membuat proyek server, mengaktifkan login Google, membuat tabel database, dan mengatur keamanan.
2
Salin kredensial proyekAmbil Project URL dan anon public key dari pengaturan proyekmu.
3
Tempel ke APP_CONFIG
const APP_CONFIG = { URL: "https://xxxxx.contoh-server.co", KEY: "eyJhbGciOi..." // anon public key };
⚠️ Simpan file setelah mengisi konfigurasi, lalu muat ulang halaman ini di browser.
